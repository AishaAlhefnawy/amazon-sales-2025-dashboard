# amazon-sales-2025-dashboard
Sales analysis dashboard built with Tableau and Excel.
# Project Overview:
This project is an interactive Tableau dashboard built to analyze Amazon sales performance and business trends.

The dashboard provides insights into:
- Sales trends over time.
- Average order value.
- Sales by category.
- Top customers.
- Product performance.
- Sales distribution by location.
- 
- # Tools Used:
- Tableau.
- Excel.

- # Key Insights:
- Electronics generated the highest sales contribution.
- Certain cities showed stronger sales performance than others.
- Top customers contributed significantly to total revenue.
- Product categories had different sales behaviors across periods.

- # Features:
- Built an interactive dashboard using Tableau.
- Explored sales data to understand patterns and trends.
- Used filters to make the dashboard easy to navigate.
- Visualized key sales metrics in a simple way.
